## 2. Gap-analyse t.o.v. Vitality Platform

### 2.1 Health / Longevity Intelligence

| Capability | Markt | Kans |
|-----------|------|------|
| Workouts tracking | Sterk | Commodity |
| Nutrition tracking | Redelijk | Commodity |
| Wearables integratie | Basis | Middel |
| Biomarkers (bloed, hormonen) | Nauwelijks aanwezig | Groot |
| Longevity modelling | Niet aanwezig | Groot |
| Holistische health score | Beperkt | Groot |

**Conclusie:** Geen enkele speler combineert fitness, voeding, biomarkers en gedrag in één geïntegreerde score.

---

### 2.2 AI / Coaching Intelligence

| Capability | Markt | Kans |
|-----------|------|------|
| Static schema’s | Standaard | Laag |
| Progress tracking | Standaard | Laag |
| Basic recommendations | Beperkt | Middel |
| AI coaching (adaptief) | Nauwelijks aanwezig | Groot |
| Context-aware coaching | Niet aanwezig | Groot |
| Predictive coaching | Niet aanwezig | Groot |

**Conclusie:** De markt biedt tracking, geen intelligente coaching.

---

### 2.3 Data Integratie / Platform

| Capability | Markt | Kans |
|-----------|------|------|
| Apple Health / Google Fit | Standaard | Laag |
| Wearables integratie | Beperkt | Middel |
| Nutrition APIs | Beperkt | Middel |
| Medical / lab data | Niet aanwezig | Groot |
| Unified data model | Niet aanwezig | Groot |
| API-first platform | Niet aanwezig | Groot |

**Conclusie:** Grote kans voor een platform met een canonical health model en event-driven architectuur.

---

### 2.4 UX / Engagement

| Capability | Markt | Kans |
|-----------|------|------|
| Workouts UI | Goed | Laag |
| Progress charts | Standaard | Laag |
| Gamification | Basis | Middel |
| Holistisch dashboard | Niet aanwezig | Groot |
| Vitality score visualisatie | Niet aanwezig | Groot |
| Actionable insights | Nauwelijks | Groot |

**Conclusie:** UX is data-heavy maar niet actiegericht.

---

### 2.5 Business Positionering

| Segment | Status | Kans |

## 3. Strategische kansen

### 3.1 Core differentiators

1. **Vitality Score**  
   - Multi-factor (fitness, voeding, slaap, biomarkers)  
   - Centrale KPI voor gezondheid  

2. **AI-driven coaching**  
   - Adaptief en context-aware  
   - Realtime aanbevelingen  

3. **Biomarker integratie**  
   - Bloedwaarden en hormonen  
   - Link met performance en herstel  

4. **Unified health data model**  
   - Canonical model  
   - Event-driven architectuur  

5. **Actionable UX**  
   - Van dashboards naar acties  
   - Focus op "wat moet ik vandaag doen"

---

## 5. Samenvatting

De huidige markt richt zich op het registreren van data. Er is een duidelijke kans voor een platform dat:

- Data integreert (fitness, voeding, slaap, biomarkers)
- Deze vertaalt naar één duidelijke score
- Acties en aanbevelingen genereert via AI

**Kerninzicht:**  
Van tracking → naar decision support en coaching.
